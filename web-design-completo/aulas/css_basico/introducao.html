CSS Básico

Linguagem de estilo que serve para definir a formatação de um documento HTML

CSS = Cascading Style Sheets

Seletores

Servem para selecionar um elemento HTML e aplicar um estilo a ele.

Exemplos: a, p, div, h1, h2, span, article, figure, img etc.

O * define uma configuração global para o documento.

Class e ID

Class é um atributo que pode ser repetido na estilização de diferentes elementos. Chamado pelo .(ponto)nomeDaClasse. -> .container

ID é um atributo que só deve ser usado uma vez, mais usado na interação com Javascript. #(hashtag)nomeDaClasse -> #paragrafo

Tanto as IDs como as Classes devem ser nomeadas conforme o seu uso, para facilitar o entendimento. Apesar dos nomes serem totalmente arbitrários ao seu uso, sim você pode colocar o que quiser como um nome de Classe ou ID.

Pseudoclasses

São ferramentas utilizadas para estilizar um elemento após uma determinada ação do usuário.

Exemplo: hover, visited, after, before, active entre outros

Para mais referências de Seletores: w3schools ou mozilla.org

Propriedades de Texto

São propriedades utilizadas para estilizar um texto

Propriedade Uso Valores aplicados
color Altera a cor do texto Hexadecimal #FFFFFF, RGB (0,0,0), RGBA(0,0,0,0.0), hsl e hsla
font-style Define o estilo do texto italic, bold, oblique
font-weight Define a espessura do caractere 100 - 900
font-size Define o tamanho do texto px (unidade fixa) ou em (unidade relativa)
line-height Define o tamanho da linha px ou em
font-family Define a fonte a ser utilizada no texto Arial, Times New, Georgia
